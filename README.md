# Ultrasonic Blind Stick using Arduino, GPS and GSM

## Overview

This project is an assistive device for visually impaired people.

It detects obstacles using ultrasonic sensors and provides alerts using buzzer.

It also integrates GPS and GSM modules to send location information during emergency.

Advanced version includes motorized wheels for automatic movement.

---

## Features

• Obstacle detection using Ultrasonic Sensor  
• GPS location tracking  
• GSM emergency alert system  
• Audio alert using buzzer  
• Automatic wheel control  

---

## Hardware Used

Arduino UNO  
Ultrasonic Sensor HC-SR04  
GPS Neo-6M  
GSM SIM900L  
Motor Driver L298N  
Buzzer  
Battery
